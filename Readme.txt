Step by Step
Step 1: Install node js: https://nodejs.org/en/
Step 2: Run comand line:  node install
Step 3: Edit file sound.text: ./data/Sound.txt
//Example
    ..Sound.txt
    one
    two
    three
//
Step 4: Open file GetSound.js:  ./GetSound.js and Edit language
//Example   
onRun('vi'); vietnamese
onRun('id'); Indonesian
onRun('en'); English
